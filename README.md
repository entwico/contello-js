# contello-js

A collection of shared JavaScript/TypeScript libraries for Contello projects.

## Packages

| Package | Description |
|---------|-------------|
| [@contello/rich-text](./packages/rich-text) | Rich text document types and helpers |

## Development

```bash
# Install dependencies
bun install

# Build all packages
bun run build

# Run tests
bun test

# Typecheck
bun run typecheck
```
